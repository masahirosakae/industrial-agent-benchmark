Industrial Agent Benchmark dataset files are released under the Apache License 2.0 unless a future dataset release explicitly states otherwise.

The repository license is provided in `LICENSE`.

## Dataset Scope

This dataset license notice applies to public benchmark data and public documentation intended for dataset publication, including:

- `data/v2/test.jsonl`
- dataset card material
- public schema documentation
- public benchmark question content

It does not grant permission to publish local artifacts that are intentionally excluded from the repository, including:

- raw model answers
- generated judge outputs
- private result directories
- unpublished run reports
- provider credentials
- internal model-name mappings

When using or redistributing the dataset, preserve the dataset name, version, license notice, and citation metadata.
