## Release Body (Markdown)

# Industrial Agent Benchmark v1.1.0-pre

Industrial Agent Benchmark v1.1-pre は、v1.0 で凍結した 90 問のベンチマークを基盤としつつ、製造業向け Agent の能力差を **より明確に検出できる評価設計** へ拡張したプレリリースです。

## ハイライト

- 問題数を **90 → 140 問** に拡張 (v1.0 既存 90 問は未変更)
- 新カテゴリ追加: Maintenance & Engineering / Risk Tradeoff / Data Integrity / Agent Safety / Structured Decision / Tool Trajectory / Human-in-the-Loop Boundary
- **Judge v2 (`judge_template_v2.md`)** を導入。`score_cap_rules` / `numeric_checks` / `generic_answer_penalty` / `structured_output_requirements` / `disallowed_answers` を機械的に適用
- **evaluation_set_v2 (30 問)** を新規追加。v1.1 追加カテゴリを重点化
- 後方互換のスキーマ拡張 (任意フィールド)
- Anonymized model evaluation で v1.1 の識別力改善を確認

## 識別力検証結果 (anonymized)

| model_id | avg_final_score |
|---|---:|
| model_a | 3.93 |
| model_b | 3.93 |
| model_c | 3.33 |
| model_d | 1.27 |

- スコアは 1〜5 点に分布 (v1.0 の 4〜5 集中を改善)
- 最高平均と最低平均の差: **2.67 点**
- score_cap 発動率 65.0% / numeric_check 失敗率平均 0.333 / generic_penalty 発動率 25.8% / critical_failure 発動 6 件
- 最も識別力が高かったカテゴリ: hil_boundary (2.00) / agent_safety (2.63) / tool_trajectory (2.63) / data_integrity (2.70)

## 非公開・非含有

本リリースには以下を含めません。

- `results_v2/` (Judge v2 採点結果、raw answers、judge inputs、judgements)
- `docs/evaluation_v2_private_notes.md` (非公開メモ)
- `results/` (v1.0 時点の非公開評価成果物)
- モデル ID 対応表、提供元名、製品名、運用名
- API キー、シークレット、認証情報

## 既知の制限

- Change Control / Compliance / Factory Management / Numeric Capacity Planning は v1.1.0 (正式版) で追加予定
- Agent Evaluation Platform (harness / MCP mock / sandbox) との統合は v2.0 で対応予定
- Judge v2 の校正は anonymized model 評価による暫定校正であり、人手レビューによる継続的キャリブレーションを推奨
