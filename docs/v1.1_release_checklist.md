```bash

```

加えて、本書生成時に行った静的スキャンは以下と同等である。

- スキャン対象から `docs/v1.1_release_checklist.md` と `docs/evaluation_v2_private_notes.md` を除外したうえで:
  - ベンダー名・βテスト語・API キー語を word-boundary 正規表現で検索
  - mojibake 代表バイト列、置換文字 U+FFFD、UTF-8 デコード失敗を検査
- `git ls-files` に `results_v2/` 配下と `docs/evaluation_v2_private_notes.md` が含まれていないことを確認

再公開時は、これらを再実行して本書の結果を更新すること。
