# Industrial Agent Benchmark v1.1 — Question Titles (Planned 90)

本ドキュメントは v1.1 で追加する 90 問の **タイトル一覧** と
「**問題で差分が出る理由**」を提示する。設計レビュー後、各問題を YAML として生成する。

各 ID 行に含める情報:
- ID
- Layer
- Category
- Difficulty (1〜5)
- Title (1 行)
- 差分が出る理由 (どのモデル能力差を測るか / どのスキーマ拡張を使うか)

凡例:
- `num` = numeric_checks 主軸
- `cap` = score_cap_rules 発火想定
- `gen` = generic_answer_penalty 主軸
- `str` = structured_output_requirements 主軸
- `dis` = disallowed_answers / critical_failure 主軸
- `defer` = Safe Deferral 主軸
- `tool` = expected_tool_sequence 主軸
- `hil` = hil_required_decisions 主軸

### スキーマ拡張活用カバレッジ

| 拡張機能 | 主軸となる問題数 | 主な配置カテゴリ |
|---|---|---|
| numeric_checks (`num`) | 約 45 問 | NCP / FM / Maint / RT |
| disallowed_answers / critical (`dis`) | 約 25 問 | DI / COMP / AS / HILB / Maint |
| Safe Deferral (`defer`) | 約 18 問 | DI / COMP / CHG / HILB |
| structured_output_requirements (`str`) | 約 22 問 | SD / AS / HILB / CHG / RT |
| expected_tool_sequence (`tool`) | 5 問 | TT 全問 |
| hil_required_decisions (`hil`) | 約 14 問 | HILB 全問 / COMP / AS / RT / DI |
| generic_answer_penalty (`gen`) | 全 90 問既定 ON | (一部 NCP 単純計算問題で OFF) |

これにより、v1.0 では区別が困難だった上位モデル間 (数値正答性 / 判断保留 / 構造化出力 / Tool 順序 / HIL 設計) の差分を、複数の独立軸で同時に観測できる構成となる。
