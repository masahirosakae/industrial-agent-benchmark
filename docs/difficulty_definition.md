# Industrial Agent Benchmark v1.0 — Difficulty Definition

本ドキュメントは難易度 1〜5 の正式定義を示す。
LLM-as-a-Judge による評価および採点重み付けの基準とする。

---

## 設計思想

- 単純な暗記・定義問題ではなく、**製造業の意思決定能力**を測る。
- レベル上昇に伴い、以下の難しさが上がる:
  1. 制約数の多さ（QCD + 安全 + 法規制）
  2. 部門横断性（営業/生産/品質/調達/経営）
  3. 不確実性（ノイズ・欠損データ・トレードオフ）
  4. 結果の重大性（軽微な改善 → リコール/労災レベル）

---

## レベル定義

### Level 1 — Basic Operator / 基礎実務

| 項目 | 内容 |
|---|---|
| 必要能力 | 標準作業 / SOPに従って単一の制約下で正しく判断する。条件整理、基本算術、明らかな異常の検知。 |
| 想定解答時間 | 5〜15 分 |
| 典型問題 | 与えられた寸法データから規格内/外を判定する。基本的な指差し確認手順を組み立てる。標準作業書に書かれた手順の整理。 |
| 評価重点 | 条件の正確な読み取り、ルール準拠 |

### Level 2 — Junior Engineer / 若手技術者

| 項目 | 内容 |
|---|---|
| 必要能力 | 単一工程・単一部門内での原因分析と改善案立案。FTA / 5Why / FMEA の基本適用。原因候補が限定的な状況下での推論。 |
| 想定解答時間 | 15〜30 分 |
| 典型問題 | 1工程内の不良増加に対する5Why。簡易PFMEAの作成。受入検査強化の基本案。 |
| 評価重点 | 因果推論、定型フレームワークの正しい適用 |

### Level 3 — Mid-level Engineer / 中堅技術者

| 項目 | 内容 |
|---|---|
| 必要能力 | 複数のKPI (品質 vs コスト、納期 vs 効率) のトレードオフ判断。部門横断調整。ノイズ・欠損データを含む状況での仮説立案と代替案提示。 |
| 想定解答時間 | 30〜60 分 |
| 典型問題 | 内示変動下の生産計画修正。複数候補のサプライヤ評価。出荷配分の優先順位付け。混流ラインの段取り設計。 |
| 評価重点 | トレードオフの定量比較、代替案の網羅、論理一貫性 |

### Level 4 — Senior Expert / 熟練専門家

| 項目 | 内容 |
|---|---|
| 必要能力 | 工場全体・サプライチェーン全体の設計判断。重大品質流出・リコール・労災に直結する意思決定。不確実性下での投資判断。長期影響の評価。 |
| 想定解答時間 | 45〜90 分 |
| 典型問題 | 市場クレーム発生時の封じ込め範囲決定。EOL通知への7年保守計画。新規工場立上げの初期流動計画。エージェント設計の責務分割。 |
| 評価重点 | リスクの安全側判断、長期影響評価、システム思考 |

### Level 5 — Factory Director / 工場長・チーフエンジニア

| 項目 | 内容 |
|---|---|
| 必要能力 | 物理法則・経営財務・法規制・組織論が複合した問題。正解が一意でなく、定量・定性両面から最適解を構築。HLEレベルの高度な複合判断。 |
| 想定解答時間 | 60〜180 分 |
| 典型問題 | 安全部品の熱処理逸脱疑いでの出荷停止範囲決定。複数工場間の生産配分。AI誤判断のガードレール設計。エージェント間競合の調停プロトコル。 |
| 評価重点 | 安全・法的責任の最優先、複合最適化、意思決定の説明可能性 |

## 難易度設定ガイドライン

新規問題作成時、以下のチェックリストで難易度を割り当てる:

- [ ] 必要部門数: 1部門 → L1-L2, 2-3部門 → L3, 4部門以上 → L4-L5
- [ ] 制約数: 1-2 → L1, 3-4 → L2-L3, 5以上 → L4-L5
- [ ] 結果の重大性: 軽微 → L1-L2, 顧客影響 → L3, 安全/法規制 → L4-L5
- [ ] データ不確実性: 完全データ → L1-L2, 欠損あり → L3, 矛盾含む → L4-L5
- [ ] 推論手数: 1-2手 → L1, 3-4手 → L2-L3, 5手以上 → L4-L5

## 難易度分布の方針 (v1.0)

| Level | 目安比率 |
|---|---|
| 1 | 5-10% |
| 2 | 15-25% |
| 3 | 30-40% |
| 4 | 20-30% |
| 5 | 10-15% |

正規分布に近い形を目標とし、極端な易/難の偏りを避ける。
