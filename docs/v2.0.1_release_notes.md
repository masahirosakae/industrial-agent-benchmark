# Industrial Agent Benchmark v2.0.1

## Documentation & Metadata Correction Release

v2.0.1 is a documentation and metadata correction release for Industrial Agent Benchmark.

This release corrects the public language policy description after external audit found that v2.0.0 was described as an English-only release while the actual dataset contains English, Japanese, and mixed-language records.

## What Changed

- Updated README language-policy wording.
- Updated English README language-policy wording.
- Updated Hugging Face Dataset Card metadata to include both `en` and `ja`.
- Updated repository dataset card metadata to include both `en` and `ja`.
- Clarified that v2.0.0 is the first stable public release, not an English-only release.
- Clarified that v2.1.0 is planned to formalize the multilingual English + Japanese architecture.
- Updated evaluation planning language guidance to follow each benchmark item's language.
- Removed the local private notes artifact from the public-release workspace.

## No Dataset Changes

v2.0.1 does not change benchmark data.

- No benchmark item changes.
- No JSONL schema changes.
- No split changes.
- No category distribution changes.
- No evaluation outputs added.

The dataset remains:

| Layer | Count |
|---|---:|
| Knowledge | 60 |
| Reasoning | 60 |
| Agent | 60 |
| Total | 180 |

## Corrected Classification

v2.0.1 should be described as a multilingual benchmark release.

The v2.0.x dataset contains English, Japanese, and mixed-language records. v2.1.0 is planned to provide a more explicit multilingual architecture, including language metadata and clearer English/Japanese task design.
